# Documentación AlphaApp


P O R T A D A 

El nombre de nuestra aplicación es Grupo Alpha , versión 1.0
fue creada en Abril de 2026. actualmente se encuentra en fase de pruebas


El siguiente documento, va dirigido a todos los usuarios de nuestra aplicación y tiene como funcionalidad la de informar a dicho usuario sobre las distintas funcionalidades de que dispone nuestra aplicación y su correcto modo de empleo.
Esta aplicación sirve para la realización de reservas en una empresa multiservicios, constituida por variedad de sectores, como son Gimnasio, Peluquería. Academia de enseñanza, centro de yoga , y es ampliable a nuevos servicios.
La aplicación constituye un conjunto único y autocontenido que no necesita de soporte exterior para su correcto funcionamiento
